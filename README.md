## Описание

Проект с настроенным Continuous Deployment (Непрерывное Развертывание) с использованием:
- GitHub Actions для CI/CD
- Webpack 5 для сборки
- Автоматический деплой на GitHub Pages

## Технологии
- Webpack 5
- Babel
- ESLint + Prettier
- Jest для тестирования
- GitHub Actions
